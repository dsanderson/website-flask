Parts Search







What is this?

As part of our research into design automation, we've discovered it's important to have easy access to computer-readable data on real-world components and parts. This allows fast and flexible early-stage exploration of design spaces, while remaining grounded in feasability. This site allows designers to specify a component in terms of the interfaces it provides (torque, speed, voltage, etc.) and keywords it contains, and gives back a CSV or JSON file that contains all the matching parts we could find scraping suppliers online.

How do I use this?

Each row above reprsents a requirement for the component, and has a textbox for keywords and a selector for values. Keywords can be a comma-separated list, in which case only one of the items in the list need match the document. Values specify the quantities that the component must specify, and determine the columns in the returned CSV. If the value selected is "none", the row requires one of the keywords to appear in the component's page. Otherwise, the keywords will need to appear "near" one of the values in the component description.
